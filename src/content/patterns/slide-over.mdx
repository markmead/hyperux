# Alpine.js {frontmatter.title}

`huxSlideOver` handles open/close state, backdrop behavior, focus trapping, and scoped lifecycle events for slide-over panels in Alpine.js. It encapsulates the same dialog accessibility model as `huxDialog`, adapted for edge-anchored panels that slide in from the left or right.

This pattern expects the Alpine Focus plugin when you use `x-trap` in your markup: [@alpinejs/focus](https://cdn.jsdelivr.net/npm/@alpinejs/focus@3.15/dist/cdn.min.js).

## API

### `huxSlideOver(config)`

Returns an Alpine data object with:

- `isOpen: boolean`
- `slideOverId: string | null`
- `placement: string`
- `isPersistent: boolean`
- `slideOverTitleId: string | null`
- `closeOnBackdrop: boolean`
- `closeOnEscape: boolean`
- `openSlideOver(): void`
- `toggleSlideOver(): void`
- `closeSlideOver(): void`
- `handleBackdropClick(): void`
- `handleEscape(): void`

_Internal helper methods are private implementation details and are not part of the supported API contract._

## Options

- `slideOverId: string` (optional)
- `placement: 'right' | 'left'` (default: `'right'`)
- `startsOpen: boolean` (default: `false`)
- `isPersistent: boolean` (default: `false`)

## Quick Start

```html
<div x-data="huxSlideOver({ slideOverId: 'settings' })">
  <button type="button" x-on:click="openSlideOver()">Open slide-over</button>

  <template x-teleport="body">
    <div x-cloak x-show="isOpen" x-on:keydown.escape.stop="handleEscape()">
      <div
        x-cloak
        x-show="isOpen"
        x-transition.opacity.duration.200ms
        aria-hidden="true"
        x-on:click="handleBackdropClick()"
      ></div>

      <div
        x-cloak
        x-show="isOpen"
        x-trap.noscroll="isOpen"
        x-transition:enter="transform transition-transform duration-300 ease-out"
        x-transition:enter-start="translate-x-full"
        x-transition:enter-end="translate-x-0"
        x-transition:leave="transform transition-transform duration-200 ease-in"
        x-transition:leave-start="translate-x-0"
        x-transition:leave-end="translate-x-full"
        role="dialog"
        aria-modal="true"
        x-bind:aria-labelledby="slideOverTitleId || null"
      >
        <h2 x-bind:id="slideOverTitleId || null">Settings</h2>
        <button type="button" x-on:click="closeSlideOver()">Close</button>
      </div>
    </div>
  </template>
</div>
```

## Common Usage Patterns

### Left-side Placement

```html
<div x-data="huxSlideOver({ slideOverId: 'nav-panel', placement: 'left' })">
  <button type="button" x-on:click="openSlideOver()">Open navigation</button>

  <template x-teleport="body">
    <div x-cloak x-show="isOpen" x-on:keydown.escape.stop="handleEscape()">
      <div
        x-cloak
        x-show="isOpen"
        x-transition.opacity.duration.200ms
        aria-hidden="true"
        x-on:click="handleBackdropClick()"
      ></div>

      <div
        x-cloak
        x-show="isOpen"
        x-trap.noscroll="isOpen"
        x-transition:enter="transform transition-transform duration-300 ease-out"
        x-transition:enter-start="-translate-x-full"
        x-transition:enter-end="translate-x-0"
        x-transition:leave="transform transition-transform duration-200 ease-in"
        x-transition:leave-start="translate-x-0"
        x-transition:leave-end="-translate-x-full"
        role="dialog"
        aria-modal="true"
        x-bind:aria-labelledby="slideOverTitleId || null"
        class="fixed top-0 left-0 h-full w-full max-w-md bg-white shadow-xl"
      >
        <h2 x-bind:id="slideOverTitleId || null">Navigation</h2>
        <button type="button" x-on:click="closeSlideOver()">Close</button>
      </div>
    </div>
  </template>
</div>
```

### Persistent Slide-over

```js
huxSlideOver({
  slideOverId: 'edit-record',
  isPersistent: true,
})
```

### Starts Open

```js
huxSlideOver({
  slideOverId: 'sidebar',
  startsOpen: true,

```

### Listening to Lifecycle Events

```html
<div
  x-data="huxSlideOver({ slideOverId: 'details' })"
  x-on:hux-slide-over:details:open.window="console.log('slide-over opened')"
  x-on:hux-slide-over:details:close.window="console.log('slide-over closed')"
>
  ...
</div>
```

## Behavior Contract

- `init()` derives `slideOverTitleId` from `slideOverId` as `${slideOverId}-title`, otherwise `null`.
- `openSlideOver()` sets `isOpen = true` and dispatches open event(s).
- `closeSlideOver()` sets `isOpen = false` and dispatches close event(s).
- `toggleSlideOver()` flips open state by delegating to `openSlideOver()` and `closeSlideOver()`.
- `isPersistent` disables both Escape and backdrop close via computed flags.
- `placement` defaults to `'right'`; use `'left'` for panels anchored to the left edge.
- Event names are scoped when `slideOverId` exists.

## Error Handling

- Slide-over methods are defensive and do not throw for repeated calls (`openSlideOver()` while open, `closeSlideOver()` while closed).
- Unknown event listeners or missing external handlers do not block internal state transitions.

## Accessibility Notes

- Keep the panel container with `role="dialog"` and `aria-modal="true"`.
- Provide a visible or screen-reader title and wire `aria-labelledby` to `slideOverTitleId`.
- Keep close controls as labeled `button` elements with `type="button"`.
- Use keyboard escape handling (`x-on:keydown.escape.stop`) and focus trapping (`x-trap`) so keyboard users remain inside the panel while open.
- Pair `x-teleport="body"` with the panel template so the slide-over renders at the document root and is never clipped by overflow-hidden ancestors.

## Notes

- Events emitted:
  - `hux-slide-over:{slideOverId}:open`
  - `hux-slide-over:{slideOverId}:close`
  - `hux-slide-over:open` and `hux-slide-over:close` when `slideOverId` is missing
- `placement` is exposed as reactive data so your markup can read it for conditional positioning classes when a single template handles both sides.
- Migration from `huxDialog`: replace `dialogId` with `slideOverId`, replace `openDialog` / `closeDialog` / `toggleDialog` with `openSlideOver` / `closeSlideOver` / `toggleSlideOver`, and update event name prefixes from `hux-dialog` to `hux-slide-over`.
